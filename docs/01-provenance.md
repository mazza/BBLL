# Proveniência

## Finalidade

Este documento registra a origem dos dados textuais utilizados como ponto de partida do projeto **Bíblia Brasileira Literal Livre (BBLL)**.

## Fonte dos textos originais

Os textos hebraico e grego empregados neste projeto têm origem em materiais do **Berean Bible** (Bible Hub / Berean Bible Translation Committee).

Em 30 de abril de 2023, os textos do Berean Bible foram oficialmente dedicados ao domínio público (CC0 1.0).[^1]

Os arquivos utilizados foram obtidos a partir de cópias preservadas no Internet Archive (web.archive.org), uma vez que o site original deixou de estar regularmente acessível.

Cópias desses arquivos encontram-se preservadas neste repositório, sob o diretório `upstream/berean/`.

## Processamento

A partir dos arquivos obtidos, foram realizadas operações técnicas de extração, reorganização e conversão para formato Markdown, com o objetivo de facilitar o trabalho posterior sobre o texto.

Os arquivos resultantes desse processamento, bem como os scripts utilizados, também são mantidos neste repositório.
