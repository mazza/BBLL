# BBLL

Bíblia Brasileira Literal Livre.

Projeto de tradução literal da Bíblia para o português brasileiro, a partir dos textos hebraico e grego.

Os dados de origem utilizados neste projeto foram obtidos a partir de materiais dedicados ao domínio público. A documentação de proveniência encontra-se em [`docs/01-provenance.md`](docs/01-provenance.md).
